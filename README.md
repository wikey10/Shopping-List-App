# 🛒 Shopping List App

A simple Shopping List app built with **Kotlin** and **Jetpack Compose** using Android Studio.

---

## ✨ Current Features

- ➕ Add products with quantity  
- 📝 Edit items in the list  
- ❌ Delete items from the list  

---
